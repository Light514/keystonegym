# Keystone Martial Arts Landing Page

A bold, Swiss-style landing page for a selective martial arts community.

## Features

- **Swiss Design Style**: High contrast, strict grid alignment, and massive typography (Archivo).
- **Interactive Sections**: Smooth scroll reveals using Framer Motion.
- **Responsive Layout**: Adapts from mobile vertical stacks to complex desktop grids.
- **Custom Components**: Buttons, Grids, and Typography components built for reuse.

## Usage

```tsx
import KeystoneLandingPage from '@/sd-components/a300fb39-f45c-4505-969c-cc980d7dee19';

function App() {
  return <KeystoneLandingPage />;
}
```

## Props

The component is currently self-contained with no external props required for the main landing page view.
